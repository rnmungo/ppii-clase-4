Layout y grilla

Containers, rows y columns

La grilla se basa en 12 columnas. Con clases responsivas podemos cambiar el ancho segun el dispositivo.

col-12 col-md-6 col-lg-3
col-12 col-md-6 col-lg-3
col-12 col-md-6 col-lg-3
col-12 col-md-6 col-lg-3

Clases importantes

  • container: centra y limita el ancho del contenido.
  • row: crea una fila para columnas.
  • col: define columnas flexibles.
  • g-3, g-4: agregan separacion entre columnas.

Utility classes

Margenes, paddings, colores, flex y texto

Las utilities permiten resolver estilos comunes directamente desde clases, sin escribir CSS para cada caso.

Spacing

Formato: {propiedad}{lado}-{valor}.

Tengo p-3 m-2
  • m-3: margen en todos los lados.
  • mt-4: margen superior.
  • px-5: padding horizontal.
  • py-2: padding vertical.

Personalizacion

Modificar colores y fuentes

Bootstrap se puede personalizar con CSS propio. En esta pagina, los colores y fuentes principales salen de variables de Bootstrap modificadas en :root.

Paleta actual

--bs-primary

--bs-secondary

--bs-body-bg

--bs-body-color

Codigo para mostrar en clase

:root {
  --bs-primary: #2952e3;
  --bs-primary-rgb: 41, 82, 227;
  --bs-secondary: #f97316;
  --bs-secondary-rgb: 249, 115, 22;
  --bs-body-bg: #f7f8fb;
  --bs-body-color: #172033;
  --bs-body-font-family: "Inter", sans-serif;
  --bs-border-radius: 8px;
}

body {
  background: var(--bs-body-bg);
}

.navbar {
  background: var(--bs-primary);
}

En Bootstrap por CDN, muchas utilities toman valores desde :root. Algunos componentes, como botones, definen variables locales; para esos casos se puede ajustar la clase del componente.

.custom-button {
  --bs-btn-bg: var(--bs-secondary);
  --bs-btn-border-color: var(--bs-secondary);
  --bs-btn-color: #fff;
}
